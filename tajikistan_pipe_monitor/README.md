# CrewAI Tender Monitor — Таджикистан, Трубы DN ≥ 400 мм

Система из 6 агентов и менеджера-координатора для автоматического поиска, анализа и структурирования тендеров по трубопроводам (водоснабжение, ирригация, канализация) на территории Таджикистана.

Система разработана для:
* **Барс** (компания-подрядчик) — участвует в тендерах на замену и прокладку трубопроводов.
* **composite.tj** (производитель) — поставляет стеклопластиковые (GRP/FRP) трубы диаметром DN 400–3000 мм.

## Архитектура агентов
1. **ScoutAgent**: Разведчик тендеров. Сканирует 10 ключевых источников.
2. **DetailsAgent**: Специалист по деталям. Извлекает номера лотов, бюджеты, сроки и условия.
3. **ContactAgent**: Охотник за контактами. Собирает контактные лица ПИУ/ПМУ и текущих подрядчиков.
4. **PipeCheckerAgent**: Технический эксперт. Фильтрует тендеры по диаметру труб (DN ≥ 400 мм) и материалам.
5. **RiskAnalystAgent**: Аналитик рисков. Оценивает проекты по 5 критериям рисков и дает рекомендации.
6. **ReportAgent**: Генератор отчетов. Собирает данные воедино и формирует Excel-реестр, резюме и контакты.
7. **ManagerAgent**: Координирует работу агентов в иерархическом процессе.

## Установка и запуск

1. Установите зависимости:
```bash
cd tajikistan_pipe_monitor
pip install -e .
```

2. Настройте файл `.env`:
Укажите ваш `ANTHROPIC_API_KEY` или `GEMINI_API_KEY`.

3. Запустите мониторинг:
```bash
python main.py
```

## Результаты
Все результаты сохраняются в папку `output/`:
* `tajikistan_pipe_registry_YYYYMMDD.xlsx` — структурированный Excel-реестр.
* `urgent_summary_YYYYMMDD.txt` — текстовое резюме на русском языке.
* `contacts_YYYYMMDD.json` — структурированная книга контактов.
